                                                # 说明

                                                本教程是最新的，使用 loader V3。

                                                你如果遇到了密码封锁（Password Blocked）,查看解决方法：

                                                Fikit Password Blocked!（22.10.13更新）

                                                相关信息

                                                ① 使用 LOADER V3 时不要打开任何其他程序，如果您在看到控制台后 LOADER 自动关闭，请尝试关闭一些应用程序，如各种杀毒软件。
                                                ② 当您单击加载程序的 .exe 时，不要使用 ALT+TAB， 等待第一个弹窗弹出后再操作。

                                                RDR2 菜单要求您的 PC 上没有任何其他防病毒软件，没有 WINDOWS DEFENDER，因为fikit在短时间内读写多次文件，一些杀毒软件可以检测到它并阻止此操作，导致游戏崩溃。
