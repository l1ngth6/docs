                              # 使用官方注入器注入(二选一)

                              如上图，点击Download Launchpad下载官方注入器

                              进入线下，看见人物之后打开注入器，输入激活码，点击inject注入

                              # 使用整合注入器注入(二选一)

                              此方法自动设置中文，自动安装数个免费lua脚本，比较方便
                              点此下载整合注入器open in new window
                              安装器可放在任意位置。
                              依次选择GTA5-Stand
                              输入Stand激活码
                              进入线下后点击注入

                              # 操控按键

                              功能键盘按键手柄按键
                              显示/隐藏菜单F4 或 小键盘+注入后插入手柄，按照提示操作
                              切换上下选项卡右shift/右ctrl 或 小键盘7/1
                              上下左右方向键 或 小键盘8246
                              确认回车 或 小键盘5
                              返回backspace 或 小键盘0
                              设置快捷键/高级选项字母o 或 小键盘3
